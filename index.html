About

A voxel-based game, made using python. This game is specifically for building purposes.

Skins

Inside the 'QuarryConstruct_v2.0\QuarryConstruct' folder,
replace the 'skin.png' file with the new skin and the 'arm_texture.png' with the new arm texture.
(Do not change anything else inside that folder or the game will not function.)
Note: The new skin should also be in 'png' format, with the same name, i.e. 'skin.png' & `arm_texture.png`.
You can use any image/texture editing software like Photoshop, Blockbench, Ms Paint, etc.
For the base skin/models look into the 'QuarryConstruct_v2.0\Skins' folder.

Texture Packs

For Texture Pack makers:

Download the 'texture_packs_sample.zip' file by clicking the download button below.
Then, using any editor of your choice, you can edit the textures.

Make sure not to rename any of image/model files.



For Texture Pack users:

After reciving the Texture pack zip file, extract in the 'QuarryConstruct_v2.0\QuarryConstruct' folder (Game Folder).
Delete the already existing 'assets' folder and rename the new texture pack folder to 'assets'.

Done! Enjoy!
